# ✨ Features

- 🎯 Create Quiz
- 🚀 Join Quiz using Quiz ID
- 📝 Attempt Unlimited Quizzes
- 📊 Score & Points System
- 📚 Solved Quiz History
- 📂 Your Created Quizzes
- 🔐 Firebase Authentication
- ☁️ Cloud Database
- ⚡ Beautiful Material UI
- 📱 Responsive Design
